## File naming pattern

**Lifestyle / brand images (apply to whole product):**
- `3-colour-rug_lifestyle_01.jpg`
- `3-colour-rug_lifestyle_02.jpg`
- `3-colour-rug_lifestyle_03.jpg`
- ... etc

**Variant-specific packshots (one per colour, optionally per size):**
- `3-colour-rug_light-grey_packshot.jpg`
- `3-colour-rug_soft-yellow_packshot.jpg`
- `3-colour-rug_soft-turquoise_packshot.jpg`

**Detail / close-up shots:**
- `3-colour-rug_detail_01.jpg`
- `3-colour-rug_detail_02.jpg`

**Family / group shots (multiple colours together):**
- `3-colour-rug_family_01.jpg`

## Rules

1. All lowercase
2. Words separated by hyphens (`-`), groups by underscores (`_`)
3. No spaces, no special chars
4. `.jpg` (not `.jpeg`, not `.JPG`)
5. Colour names match the HAY SIMPLE COLOUR field exactly (lowercased, hyphenated)
