## File naming pattern

**Variant packshots:** `loop-bin_8l_white_packshot.jpg`, `loop-bin_26l_soft-mint_packshot.jpg`...

**Detail:** `loop-bin_detail_01.jpg`

## Rules

Same as 3-colour-rug. Capacity goes as `8l`, `26l`, `58l` (lowercase, no space).
